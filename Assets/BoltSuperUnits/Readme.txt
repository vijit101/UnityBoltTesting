+ Bolt Super Units is a collection of 100+ Super Units for Bolt - A Visual Scripting Package.

+ How to use the package.
Step 1: Install Bolt.
    - Bolt package: https://assetstore.unity.com/packages/tools/visual-scripting/bolt-163802.
    - Install guide: https://docs.unity3d.com/2019.3/Documentation/Manual/bolt-installation.html.
Step 2: Install this package.
Step 3: Using super units.
    - Just drag the super unit in the bolt graph to use them.
    - A simple tutorial: https://www.youtube.com/watch?v=o3GTDHBbUuQ&t=5s.

+ The full description of all super unit locate on https://docs.google.com/spreadsheets/d/1gtRQdpF-BGgDUHXaod6slamdrJG0iXbgb9oMxW1Lsqo/edit?usp=sharing.

+ Some super units require some types that don't included in Bolt by default. You have to add those types.
    - In the above google spreadsheet, the "Dependencies" column are the type that needs to add for that super unit working.
    + How to add new types to bolt.
        - You have to update Bolt Unit Options by using Bolt Options Wizard.
        - In Unity Editor, select menu Tools -> Bolt -> Unit Options Wizard -> Next.
        - Click the "+" symbol to add a new type. Search by name to get the suitable type.
